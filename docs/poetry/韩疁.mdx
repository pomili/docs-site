# 韩疁

## 词人简介
韩疁，生卒年不详，字子耕，号萧闲，南宋词人。其词流传甚少，《全宋词》仅存数首。词风清疏淡雅，善于通过日常生活细节传达深沉的情感。

## 高阳台·频听银签

**原文：**
> 频听银签，重燃绛蜡，年华衮衮惊心。饯旧迎新，能消几刻光阴。老来可惯通宵饮，待不眠、还怕寒侵。掩清尊、多谢梅花，伴我微吟。
>
> 邻娃已试春妆了，更蜂腰簇翠，燕股横金。勾引东风，也知芳思难禁。朱颜那有年年好，逞艳游、赢取如今。恣登临、残雪楼台，迟日园林。

**创作背景：**
此词写于除夕守岁之夜。词人年岁已高，在辞旧迎新之际，感慨时光流逝、青春不再，同时亦有珍惜当下、尽情游赏的旷达之心。

**词意简析：**
上片写除夕守岁场景，"频听银签，重燃绛蜡"以漏箭相催、红烛重燃写尽长夜漫漫，"年华衮衮惊心"直抒对时光流逝的惊惧之心。词人说自己年老了已不惯通宵饮酒，不睡又怕寒，幸有梅花相伴微吟，于寂寞中见清雅。下片由邻家少女试春妆写起，引出对青春的感叹，"朱颜那有年年好"道出容颜易老之悲，但随即化悲为旷，"恣登临、残雪楼台，迟日园林"表达了及时行乐、不负春光的人生态度。
