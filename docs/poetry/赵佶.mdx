# 赵佶

## 词人简介
赵佶（1082—1135），即宋徽宗，北宋第八位皇帝。在位期间政治上昏庸，重用蔡京等奸臣，崇奉道教，导致朝政荒废，最终引发靖康之变。但在艺术上造诣极高，书法创"瘦金体"，工花鸟画，亦擅诗词。1127年被金兵俘虏北行，1135年死于五国城（今黑龙江依兰）。其词以亡国被掳后所作最为感人，王国维《人间词话》称其《宴山亭》为"血书"之作。

## 宴山亭·北行见杏花

**原文：**
> 裁剪冰绡，轻叠数重，淡著燕脂匀注。新样靓妆，艳溢香融，羞杀蕊珠宫女。易得凋零，更多少、无情风雨。愁苦。问院落凄凉，几番春暮？
>
> 凭寄离恨重重，者双燕何曾，会人言语？天遥地远，万水千山，知他故宫何处？怎不思量？除梦里、有时曾去。无据。和梦也、新来不做。

**创作背景：**
此词作于靖康二年（1127年），宋徽宗赵佶与其子钦宗赵桓被金兵俘虏、押往北方五国城途中。赵佶见杏花盛开，触景生情，百感交集，写下这首如泣如诉的词作。以杏花的凋零自喻亡国之痛，是全词的基调。

**词意简析：**
上片以杏花为吟咏对象——花瓣如同裁剪的洁白丝绸，轻叠数重，均匀染抹着淡淡的胭脂，艳丽非凡。然而美丽如此易得凋零，更兼无情风雨摧残。"愁苦"二字直抒胸臆，以"院落凄凉，几番春暮"感叹自己因居异域的凄凉处境。下片转入直接抒写内心离恨——欲托双燕传递重重离恨，奈何燕子何曾能懂人言？天遥地远，不知故宫在何处，只能在梦中偶尔归去。最后"和梦也、新来不做"，连梦境都不再有了，绝望之情达到顶点。全词以杏花之凋零对应国家之破灭，字字血泪，感人至深。
