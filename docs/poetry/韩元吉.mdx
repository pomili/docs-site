# 韩元吉

## 词人简介
韩元吉（1118—1187），字无咎，号南涧，南宋词人。官至吏部尚书，为南宋中兴名臣之一。其词风格豪放与婉约兼擅，有《南涧诗余》传世。

## 六州歌头·东风著意

**原文：**
> 东风著意，先上小桃枝。红粉腻，娇如醉，倚朱扉。记年时。隐映新妆面，临水岸，春将半，云日暖，斜桥转，夹城西。草软莎平，跋马垂杨渡，玉勒争嘶。认蛾眉凝笑，脸薄拂燕支。绣户曾窥。恨依依。
>
> 共携手处，香如雾，红随步，怨春迟。消瘦损，凭谁问。只花知。泪空垂。旧日堂前燕，和烟雨，又双飞。人自老，春长好，梦佳期。前度刘郎，几许风流地，花也应悲。但茫茫暮霭，目断武陵溪。往事难追。

**创作背景：**
此词为追忆旧日恋情之作。词人重回故地，见桃花依旧而人事已非，触景生情，追念当年与佳人携手同游的美好时光，抒写物是人非、往事难追的深沉感慨。

**词意简析：**
上片从东风催开桃花写起，由花及人，追忆当年在水岸斜桥旁初见佳人的情景：她红粉含娇、依门而立，芳草如茵、垂杨渡口、骏马争嘶，两人眉目传情，从绣户中窥见她的笑靥。下片写如今重游故地，携手之处香已如雾，春色依旧而人已消瘦憔悴。堂前旧燕双双飞回，而人已老去，春长好而梦难圆。以刘禹锡重游玄都观自喻，感叹花若有知也应为此悲伤。结尾以茫茫暮霭中望断桃源之路作结，往事一去难追，余韵悠长。

## 好事近·凝碧旧池头

**原文：**
> 凝碧旧池头，一听管弦凄切。多少梨园声在，总不堪华发。
>
> 杏花无处避春愁，也傍野烟发。惟有御沟声断，似知人呜咽。

**创作背景：**
此词为韩元吉在汴京沦陷后所作。凝碧池原是唐代宫中池名，此处借指北宋故宫。词人听到故国梨园旧曲，勾起无限亡国之痛和身世之感。

**词意简析：**
此词短小精悍，意蕴深远。开篇以凝碧池起兴，管弦凄切之声令人不忍卒听。多少梨园旧曲仍在流传，而听者已是满头华发，国破家亡的沧桑感油然而生。下片以杏花起笔，花无处躲避春愁，只能在野烟中寂寞开放——以花喻人，写沦陷区人民的悲苦。结尾写御沟水声已断，仿佛它也懂得人的呜咽，以景结情，含蓄深沉，亡国之痛尽在不言中。
