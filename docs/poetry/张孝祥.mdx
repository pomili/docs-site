# 张孝祥

## 词人简介
张孝祥（1132—1169），字安国，号于湖居士，南宋著名词人。绍兴二十四年进士第一名（状元），历任中书舍人、建康留守等职。其词风格豪放雄迈，上承苏轼，下启辛弃疾，有《于湖词》传世。

## 六州歌头·长淮望断

**原文：**
> 长淮望断，关塞莽然平。征尘暗，霜风劲，悄边声。黯销凝。追想当年事，殆天数，非人力。洙泗上，弦歌地，亦膻腥。隔水毡乡，落日牛羊下，区脱纵横。看名王宵猎，骑火一川明。笳鼓悲鸣。遣人惊。
>
> 念腰间箭，匣中剑，空埃蠹，竟何成。时易失，心徒壮，岁将零。渺神京。干羽方怀远，静烽燧，且休兵。冠盖使，纷驰骛，若为情。闻道中原遗老，常南望、翠葆霓旌。使行人到此，忠愤气填膺。有泪如倾。

**创作背景：**
此词作于南宋孝宗隆兴年间，张孝祥任建康留守时。当时南宋与金国对峙，北伐失败后朝廷主和派占据上风，词人登楼远望淮河前线，感时伤世，写下了这首慷慨悲壮的词篇。

**词意简析：**
此词是南宋爱国词的代表作之一。上片写词人极目远望淮河前线，关塞荒凉、征尘暗天、边声悄然，追想当年靖康之变，洙泗弦歌之地如今沦为膻腥之所。隔河便是金人的毡帐和牧区，看敌军宵猎的火把照得满川通明，笳鼓悲鸣令人心惊。下片写腰间箭、匣中剑空被尘埃蠹蚀，壮志难酬，岁月空逝，神京渺远。朝廷以"干羽"为名放弃北伐，使臣往来奔走求和，怎不令人难堪。听说中原遗老常向南盼望王师的旌旗，令有志之士到此忠愤填膺，泪如雨下。全词慷慨悲凉，豪放中见沉痛。

## 念奴娇·过洞庭

**原文：**
> 洞庭青草，近中秋，更无一点风色。玉鉴琼田三万顷，著我扁舟一叶。素月分辉，明河共影，表里俱澄澈。悠然心会，妙处难与君说。
>
> 应念岭海经年，孤光自照，肝肺皆冰雪。短发萧骚襟袖冷，稳泛沧浪空阔。尽吸西江，细斟北斗，万象为宾客。扣舷独啸，不知今夕何夕。

**创作背景：**
此词作于宋孝宗乾道二年（1166年），张孝祥因受政敌谗害被罢官，从桂林北归途经洞庭湖时所作。词人面对洞庭湖的浩渺壮阔，将自己高洁的人格与湖光月色融为一体，表达出超然物外的豁达胸怀。

**词意简析：**
上片写洞庭湖中秋前夕的景色，风平浪静，三万顷湖面如美玉般的明镜，自己的一叶扁舟飘荡其中。素月与银河的光辉交相映照，上下一片澄澈。词人心领神会这妙不可言的境界。下片转入内心独白，回想自己在岭南的岁月，孤光自照，肝胆如冰雪般洁白。如今短发萧骚、襟袖生寒，却稳泛扁舟于沧浪空阔之中。词人豪情勃发，要吸尽西江之水，以北斗为杯细细斟饮，邀天地万象为座上宾客。扣舷独啸，浑然忘却身在何时。全词意境空灵澄澈，格调高远超逸，是宋代豪放词中的巅峰之作。
