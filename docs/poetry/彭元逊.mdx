# 彭元逊

## 词人简介
彭元逊，字巽吾，庐陵（今江西吉安）人，宋末元初词人。与刘辰翁交好，常有唱和。其词多写羁旅之思、身世之感，风格清婉深沉。

## 疏影·寻梅不见（江空不渡）

**原文：**
> 江空不渡，恨蘼芜杜若，零落无数。远道荒寒，婉娩流年，望望美人迟暮。风烟雨雪阴晴晚，更何须、春风千树。尽孤城、落木萧萧，日夜江声流去。
>
> 日晏山深闻笛，恐他年流落，与子同赋。事阔心违，交淡媒劳，蔓草沾衣多露。汀洲窈窕余醒寐，遗佩浮沈沣浦。有白鸥、淡月微波，寄语逍遥容与。

**创作背景：**
此词以"寻梅不见"为题，表面写寻梅不遇，实则寄托对理想境界的追寻和对身世飘零的感慨。词人借梅花之清高绝俗来抒写自己的情怀。

**词意简析：**
上片写寻梅不遇的过程：江天空阔，蘼芜杜若零落无数，却独不见梅。远道荒寒中，流年婉娩而逝，望美人（梅花）如望迟暮佳人，终不可得。风烟雨雪、阴晴不定，又何须春风千树。孤城落木萧萧，江声日夜流去，铺写出苍茫寂寥之境。下片转入更深层的寄托：山深闻笛，恐他年与梅同流落江湖。"事阔心违，交淡媒劳"写人生际遇多舛，理想难遂。汀洲余醒寐中遗佩浮沉，末以白鸥淡月、微波寄语作结，以逍遥容与自慰。全词借寻梅写心，寄托遥深。

## 六丑·杨花

**原文：**
> 似东风老大，那复有、当时风气。有情不收，江山身是寄。浩荡何世。但忆临官道，暂来不住，便出门千里。痴心指望回风坠。扇底相逢，钗头微缀。他家万条千缕，解遮亭障驿，不隔江水。
>
> 瓜洲曾舣。等行人岁岁。日下长秋，城乌夜起。帐庐好在春睡。共飞归湖上，草青无地。愔愔雨、春心如腻。欲待化、丰乐楼前，青门都废。何人念、流落无几。点点抟作，雪绵松润，为君裛泪。

**创作背景：**
此词咏柳絮（杨花），与苏轼《水龙吟·次韵章质夫杨花词》同为咏杨花名篇。词人以杨花之飘零无依自喻，寄寓身世飘零、流落无依之痛。

**词意简析：**
上片以东风老大起笔，写暮春时节杨花飘零，不复当初风气。杨花有情而无所依托，以江山为寄身之所，浩荡不知所之。回忆临官道时，暂来即去，出门千里，痴心指望回风携归。曾在扇底钗头短暂驻足，而万条千缕的垂杨遮亭障驿，却隔不断江水东流。下片写瓜洲曾系舟等待，岁岁年年。日下长秋，城乌夜起，帐庐春睡中梦见与杨花共飞湖上。愔愔雨中春心如腻，欲飞至丰乐楼前而青门都已荒废。末叹无人念此流落无依的杨花，只有点点化作雪绵，为君拭泪。全词以杨花自喻，写尽飘零之感。
