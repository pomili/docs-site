# 韩缜

## 词人简介
韩缜（1019—1097），字玉汝，原籍真定灵寿（今属河北），徙居雍丘（今河南杞县）。北宋庆历二年（1042年）进士，累官知枢密院事、尚书右仆射兼中书侍郎，出知颍昌府。以太子太保致仕，卒谥庄敏。《全宋词》仅录其词一首，但此词足以使其在词学史上留名。

## 凤箫吟·锁离愁

**原文：**
> 锁离愁、连绵无际，来时陌上初熏。绣帏人念远，暗垂珠露，泣送征轮。长行长在眼，更重重、远水孤云。但望极楼高，尽日目断王孙。
>
> 销魂。池塘别后，曾行处、绿妒轻裙。恁时携素手，乱花飞絮里，缓步香茵。朱颜空自改，向年年、芳意长新。遍绿野，嬉游醉眼，莫负青春。

**创作背景：**
此词为韩缜唯一传世的词作，借咏芳草以寄托别离情绪。全词以连绵无际的春草起兴，将离愁别绪与芳草意象紧密结合，借女子口吻抒写思妇对远行人的深切思念，是宋代咏物抒情词中的佳作。

**词意简析：**
上片以芳草起兴——离愁如同连绵无际的春草，当初与心上人同游时正值陌上初熏。接下来写女子念远，暗垂泪珠，泣送征轮远去。人已远去，她还在登楼极目远望，但只能看到远水孤云，望穿秋水也难觅游子踪影。下片追忆往昔的欢愉——池塘别后再也未能同游，当年曾携手漫步花丛柳絮之中，绿草都妒忌她的罗裙。如今朱颜虽改，但芳心依旧。最后以"莫负青春"作结，看似开朗，实则愈显凄婉，因为逝去的时光永不再来。
