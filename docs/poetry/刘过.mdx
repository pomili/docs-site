# 刘过

## 词人简介

刘过（1154—1206），字改之，号龙洲道人，吉州太和（今江西泰和）人。南宋豪放派词人，与刘克庄、刘辰翁并称"三刘"。一生四次应举不中，布衣终身，流落江湖，却与辛弃疾、陆游等交往甚密。其词学辛弃疾，豪放激越，风格狂逸，但也有婉约清丽之作。著有《龙洲词》。

## 唐多令·芦叶满汀洲

**原文：**
> 芦叶满汀洲，寒沙带浅流。二十年、重过南楼。柳下系船犹未稳，能几日，又中秋。
>
> 黄鹤断矶头，故人今在否？旧江山、浑是新愁。欲买桂花同载酒，终不似，少年游。

**创作背景：**
此词约作于宋宁宗嘉泰四年（1204年），刘过时年约五十岁。二十年前，词人曾游武昌安远楼（南楼），与友人欢聚；二十年后重过此地，故地重游而人事已非。此时南宋国势日衰，金兵威胁不断，词人感怀身世与国事，将个人之愁与家国之愁融于一词。

**词意简析：**
上片写景叙事：芦叶满汀、寒沙浅流，一派萧瑟秋色；"二十年重过南楼"点出时距之长，"柳下系船犹未稳"写行色匆匆，"能几日又中秋"感叹时光飞逝。下片抒情："黄鹤断矶头"暗用崔颢诗意，"故人今在否"问得苍凉；"旧江山浑是新愁"将个人的故人之思升华为家国之恨——江山依旧，却处处是新的哀愁。末句"欲买桂花同载酒，终不似，少年游"是全词最动人的句子：虽想重温旧日欢乐，但少年时的那份心境早已不再。此词以朴实的语言写出了中年人对青春不再、国事日非的双重感慨。
