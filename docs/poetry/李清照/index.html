李清照

词人简介​

李清照（1084年－约1155年），号易安居士，齐州章丘（今山东济南章丘）人。宋代婉约派代表词人，被誉为"千古第一才女"。出身书香门第，父亲李格非为"苏门后四学士"之一，丈夫赵明诚是金石学家。早期生活优裕，词作多写闺中闲愁与离别相思；南渡后，国破家亡、丈夫病逝，词风转为沉郁悲凉，多抒写身世飘零之痛与故国之思。其词善用白描，语言清丽，音律谐美，自成一格，人称"易安体"。有《漱玉词》传世。

如梦令·昨夜雨疏风骤​

原文：

昨夜雨疏风骤，浓睡不消残酒。试问卷帘人，却道海棠依旧。知否，知否？应是绿肥红瘦。

创作背景： 此词为李清照早期作品。词人宿醉醒来，一夜风雨之后，关切窗外海棠花是否凋零，以小令形式委婉表达惜花伤春之情。

词意简析： 全词仅三十三字，却写出无限深婉。起首两句交代昨夜风雨与宿醉，接以与侍女的问答牵引出对花事的关切。"知否，知否"叠字连用，急切中透着不甘；"绿肥红瘦"四字以对仗手法写暮春之景，绿叶肥茂、红花凋瘦，语浅意深，为千古名句。全词以惜花写伤春，娇柔中见清逸。

凤凰台上忆吹箫·香冷金猊​

原文：

香冷金猊，被翻红浪，起来慵自梳头。任宝奁尘满，日上帘钩。生怕离怀别苦，多少事、欲说还休。新来瘦，非干病酒，不是悲秋。

休休，这回去也，千万遍《阳关》，也则难留。念武陵人远，烟锁秦楼。惟有楼前流水，应念我、终日凝眸。凝眸处，从今又添，一段新愁。

创作背景： 此词约作于李清照婚后不久，丈夫赵明诚离家远游之际。词人独守空闺，离愁难遣，将满怀别苦倾注笔端。

词意简析： 上片以"慵"字为词眼，从香炉已冷、锦被乱摊、懒于梳头、镜奁尘满等细节，层层铺写离愁之深。"新来瘦，非干病酒，不是悲秋"三句，以否定手法反衬出唯有"离怀别苦"才是消瘦的真正原因。下片直抒离别之痛——即使唱千万遍《阳关》也留不住远行之人。结尾以楼前流水见证自己终日凝眸的痴情，从今更添"一段新愁"，余韵悠长，极为动人。

醉花阴·薄雾浓云愁永昼​

原文：

薄雾浓云愁永昼，瑞脑销金兽。佳节又重阳，玉枕纱厨，半夜凉初透。

东篱把酒黄昏后，有暗香盈袖。莫道不销魂，帘卷西风，人比黄花瘦。

创作背景： 此词为重阳节时李清照思念远游在外的丈夫赵明诚所作。据载，赵明诚读后叹赏不已，闭门三日，废寝忘食，欲作一首胜之，终不及。

词意简析： 上片以"薄雾浓云"烘托白日的沉闷与漫长，瑞脑香烟袅袅，更添孤寂。重阳佳节本应与亲人团聚，词人却独卧纱帐，夜半凉意初透，身心俱寒。下片写黄昏后独自东篱把酒，暗香盈袖而人不在侧。"莫道不销魂，帘卷西风，人比黄花瘦"为千古名句——以凄冷的西风中瘦菊比喻为相思而憔悴的人，形神兼备，比喻精妙绝伦。

声声慢·寻寻觅觅​

原文：

寻寻觅觅，冷冷清清，凄凄惨惨戚戚。乍暖还寒时候，最难将息。三杯两盏淡酒，怎敌他、晚来风急？雁过也，正伤心，却是旧时相识。

满地黄花堆积，憔悴损，如今有谁堪摘？守着窗儿，独自怎生得黑？梧桐更兼细雨，到黄昏、点点滴滴。这次第，怎一个愁字了得！

创作背景： 此词为李清照南渡之后、丈夫赵明诚去世后的晚年名作。词人流落江南，国破家亡，孤苦伶仃，在一个秋日黄昏，将满怀悲愁凝结于词中。

词意简析： 开篇七组十四个叠字为千古奇笔，"寻寻觅觅"写心神无主，"冷冷清清"写环境之寂，"凄凄惨惨戚戚"写内心之悲，三句层层递进，将愁绪推至极处。此后以淡酒、急风、过雁、黄花、梧桐、细雨等冷秋意象，反复渲染孤寂凄凉的氛围。末句"怎一个愁字了得"以一当十，言有尽而意无穷，将全篇愁情推向极致，堪称千古绝唱。

念奴娇·萧条庭院​

原文：

萧条庭院，又斜风细雨，重门须闭。宠柳娇花寒食近，种种恼人天气。险韵诗成，扶头酒醒，别是闲滋味。征鸿过尽，万千心事难寄。

楼上几日春寒，帘垂四面，玉阑干慵倚。被冷香消新梦觉，不许愁人不起。清露晨流，新桐初引，多少游春意。日高烟敛，更看今日晴未。

创作背景： 此词为李清照早期春闺独处怀人之作，大约作于寒食节前后。丈夫赵明诚外出游历，词人独守深闺，愁绪万千。

词意简析： 上片从"萧条庭院"和"斜风细雨"写起，渲染出春寒料峭的压抑氛围。"宠柳娇花"四字极新巧，以人之情态写柳与花。险韵诗成、扶头酒醒，看似闲雅，实则无奈——万千心事欲寄无凭。下片写连日春寒中帘幕低垂、栏杆慵倚的百无聊赖。结尾以"清露晨流，新桐初引"一转，天气初晴，微露游春之意，却又以"更看今日晴未"作结，含蓄蕴藉，别有余韵。

永遇乐·落日熔金​

原文：

落日熔金，暮云合璧，人在何处。染柳烟浓，吹梅笛怨，春意知几许。元宵佳节，融和天气，次第岂无风雨。来相召、香车宝马，谢他酒朋诗侣。

中州盛日，闺门多暇，记得偏重三五。铺翠冠儿，捻金雪柳，簇带争济楚。如今憔悴，风鬟霜鬓，怕见夜间出去。不如向、帘儿底下，听人笑语。

创作背景： 此词为李清照晚年流寓临安（今杭州）时的元宵节感怀之作。南渡后，词人历经丧夫之痛与流离之苦，元宵佳节引发她对故国往昔的深切追忆。

词意简析： 上片写眼前元宵景象：落日如熔金般灿烂，暮云如合璧般瑰丽，春意渐浓。然而面对诗朋酒友的相召，词人却一一谢绝——"次第岂无风雨"一句，既存天有不测之隐忧，更寓身世无常之沉痛。下片追忆当年汴京元宵盛况：翠冠金柳，盛装出游，青春风姿楚楚动人。而今风鬟霜鬓，憔悴不堪，只能躲在帘儿底下"听人笑语"。以他人之乐衬自身之悲，以平淡之语写深沉之痛，读来令人潸然。

浣溪沙·髻子伤春慵更梳​

原文：

髻子伤春慵更梳，晚风庭院落梅初。淡云来往月疏疏。

玉鸭熏炉闲瑞脑，朱樱斗帐掩流苏。通犀还解辟寒无。

创作背景： 此词为李清照早期作品，写贵族女子春日独处的寂寥之情，风格婉约细腻。

词意简析： 上片由人及景——"髻子伤春慵更梳"开篇即点出因伤春而无心装扮的慵懒心态，晚风中庭院梅花初落，淡云疏月，构成一幅凄清画面。下片转入室内，玉鸭熏炉中的瑞脑香闲置不燃，朱樱斗帐遮掩着流苏，华美物件的静止反衬出主人公内心的孤寂。末句"通犀还解辟寒无"以问句作结——这通犀角能为我辟除春寒吗？实则问的是能解心中之寒否。辞意婉转而含蓄，怨而不怒，为易安早期婉约词风的代表之作。
