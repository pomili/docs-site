# 陆游

## 词人简介
陆游（1125—1210），字务观，号放翁，越州山阴人。南宋伟大的爱国诗人，一生作诗近万首，词亦卓然成家。其词兼具豪放与婉约之长，既有慷慨悲歌的报国之志，也有深婉缠绵的爱情之作，在南宋词坛独树一帜。

## 卜算子·咏梅

**原文：**
> 驿外断桥边，寂寞开无主。已是黄昏独自愁，更着风和雨。
> 无意苦争春，一任群芳妒。零落成泥碾作尘，只有香如故。

**创作背景：**
此词作于陆游晚年，他一生力主抗金北伐，却屡遭主和派排挤打击，壮志难酬。他以梅花自喻，借咏梅来抒发自己虽身处困境、备受冷落，但始终坚守节操、不改初心的情怀。

**词意简析：**
上片写梅花所处的恶劣环境：在驿站外的断桥边，寂寞开放而无人欣赏，黄昏中独自愁苦，更有风雨摧残。这梅花的处境正是陆游自身命运的写照——被排挤在朝廷之外，孤寂凄凉。下片转入议论：梅花无意与百花争春，任凭群芳嫉妒，即便凋零飘落入泥被踏成尘土，那清香依然如故。以梅花的清香不死，象征诗人坚贞不屈的品格。全词以物喻人，托物言志，将咏物与自咏完美结合，是咏梅词中不可逾越的巅峰之作。
