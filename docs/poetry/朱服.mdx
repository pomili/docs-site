# 朱服

## 词人简介
朱服（1048—？），字行中，湖州乌程（今浙江湖州）人。北宋熙宁六年（1073年）进士，累官至知州、集贤殿修撰。后坐事被贬，流落多年。其词仅存《渔家傲》一首，被选入《宋词三百首》。词中"而今乐事他年泪"一句为世所传诵。

---

## 渔家傲·小雨纤纤风细细

**原文：**
> 小雨纤纤风细细，万家杨柳青烟里。恋树湿花飞不起，愁无比，和春付与东流水。
>
> 九十光阴能有几？金龟解尽留无计。寄语东阳沽酒市，拚一醉，而今乐事他年泪。

**创作背景：**
此词别题《春词》，约作于词人贬谪期间。上片以暮春雨景入题，抒写惜春之愁；下片感叹春光短暂、人生易逝，流露出贬谪后借酒浇愁的愤懑与无奈。结尾"而今乐事他年泪"一句意蕴深长，为全词最精彩的名句。

**词意简析：**
上片写暮春雨景——细雨纤纤，微风细细，万家的杨柳都笼罩在青烟一般的雨雾之中。花被雨打湿贴在枝头，飞不起来，无限愁绪与春天一同付给了东流之水。下片由惜春转到伤己——春天九十天的光阴能有几多？即使像贺知章那样解下金龟换酒，也留不住春光。"寄语东阳沽酒市，拚一醉"是借酒遣怀，而"而今乐事他年泪"以一句涵盖双重时间——今日看似快乐的事，到了他年回想起来都会变成辛酸的泪。一种人生无常、今日之乐即是明日之悲的深沉感慨，令人回味不尽。
