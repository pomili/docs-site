# 李重元

## 词人简介
李重元，生卒年及生平事迹不详，约活动于北宋后期。其传世词作极少，《全宋词》仅录《忆王孙》四首，分别写春、夏、秋、冬四季闺情。词风婉约含蓄，意境深远。

---

## 忆王孙·萋萋芳草忆王孙

**原文：**
> 萋萋芳草忆王孙。柳外楼高空断魂。杜宇声声不忍闻。欲黄昏。雨打梨花深闭门。

**创作背景：**
此词为《忆王孙》四季词中的春词。词人以暮春景色为背景，写一位女子在春日思念远方行人（王孙）的愁绪。全词意境深远，以景写情，不言愁而愁自现。

**词意简析：**
首句"萋萋芳草忆王孙"化用《楚辞·招隐士》"王孙游兮不归，春草生兮萋萋"，以芳草萋萋引出对远方之人的思念。"柳外楼高空断魂"写登楼远望只见柳色而无归人，空自销魂。第三句"杜宇声声不忍闻"借杜鹃啼血之声更添悲切。末二句"欲黄昏，雨打梨花深闭门"是全词的点睛之笔——黄昏将近、梨花被雨打落、深闭的院门，三个意象叠合，将孤独与思念推向了极致，含蓄蕴藉，余味无穷。
