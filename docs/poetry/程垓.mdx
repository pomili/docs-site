# 程垓

## 词人简介
程垓，字正伯，号书舟，生卒年不详，南宋词人。眉山（今属四川）人，苏轼中表兄弟程正辅之孙。其词风格婉约清丽，有《书舟词》传世。

---

## 水龙吟·夜来风雨匆匆

**原文：**
> 夜来风雨匆匆，故园定是花无几。愁多怨极，等闲孤负，一年芳意。柳困桃慵，杏青梅小，对人容易。算好春长在，好花长见，原只是、人憔悴。
>
> 回首池南旧事，恨星星、不堪重记。如今但有，看花老眼，伤时清泪。不怕逢花瘦，只愁怕、老来风味。待繁红乱处，留云借月，也须拚醉。

**创作背景：**
此词写词人暮年对春光流逝、人生易老的深沉感慨。一夜风雨过后，想到故园花朵必然凋零无几，由此引发对年华老去的无限伤感。

**词意简析：**
上片从夜来风雨起笔，料想故园花已无几，愁怨之中辜负了一年芳意。柳困桃慵、杏青梅小，春光对人如此轻易便逝去。词人说算来好春长在、好花长见，只是人自己憔悴了而已——这句话表面通达，实则深含悲凉。下片回首往事，池南旧事已如星星点点不堪重记，如今只有老眼看花、伤时落泪。不怕花瘦，只怕老来心境凄凉。结尾说要趁繁红尚在之时，留云借月，拚却一醉——在放达之中更见无奈与悲凉。
