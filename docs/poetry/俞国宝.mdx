# 俞国宝

## 词人简介

俞国宝（生卒年不详），号醒庵，临川（今江西抚州）人。南宋淳熙年间太学生，以词名世。据《武林旧事》载，宋高宗赵构游西湖时见其题词于酒肆屏风上，大为赞赏，即日命解褐授官。其《风入松》一词因此名满天下。存词不多，风格流丽清婉。

---

## 风入松·一春长费买花钱

**原文：**
> 一春长费买花钱，日日醉湖边。玉骢惯识西湖路，骄嘶过、沽酒楼前。红杏香中箫鼓，绿杨影里秋千。
>
> 暖风十里丽人天，花压鬓云偏。画船载取春归去，余情付、湖水湖烟。明日重扶残醉，来寻陌上花钿。

**创作背景：**
此词题于西湖酒肆屏风上。据周密《武林旧事》记载，淳熙年间，太上皇宋高宗赵构游西湖时偶入酒肆，见此词，大为赞赏，问何人所题，得知是太学生俞国宝，即日命解褐授官。原作末句为"明日再携残酒"，高宗改为"明日重扶残醉"，一字之改，意境大不相同。

**词意简析：**
上片写日日买醉西湖的生活：一春的买花钱都花在了游湖上，连马都熟悉了西湖的路，经过酒楼前总要嘶叫。"红杏香中箫鼓，绿杨影里秋千"两句以工整的对偶写出了西湖春日的繁华与旖旎——花香气中传来箫鼓之声，绿杨荫里秋千荡漾，声色俱佳。下片写游春之人与游后之意："暖风十里丽人天"化用杜甫诗意，写西湖春日的温暖与游女之盛；"画船载取春归去，余情付湖水湖烟"写暮归，将未尽的情意付与湖光水色。末句"明日重扶残醉，来寻陌上花钿"写得风流蕴藉——明日还来，既是寻昨日的花钿，更是寻昨日的欢乐。全词写出了南宋西湖的繁华景象，也是士大夫春日游乐生活的真实写照。
