# 宋祁

## 词人简介
宋祁（998—1061），字子京，安州安陆（今属湖北）人，后徙开封雍丘（今河南杞县）。北宋天圣二年（1024年）与兄宋庠同榜进士，累官至工部尚书、翰林学士承旨。与欧阳修同修《新唐书》。其词语言工丽，善于炼字，"红杏枝头春意闹"一句因着一"闹"字而境界全出，世称"红杏尚书"。卒谥景文。

## 木兰花·东城渐觉风光好

**原文：**
> 东城渐觉风光好，縠皱波纹迎客棹。绿杨烟外晓寒轻，红杏枝头春意闹。
>
> 浮生长恨欢娱少，肯爱千金轻一笑？为君持酒劝斜阳，且向花间留晚照。

**创作背景：**
此词为宋祁泛舟春游之作，写春日东城游赏所见所感。上片描绘春光灿烂之景，下片由乐生感，感叹人生苦短、欢娱太少，表达了珍惜春光、及时行乐的情怀。词人时任工部尚书，词中透露出文人雅士对美好生活的热爱与对时光易逝的深沉感慨。

**词意简析：**
上片以精工之笔描绘春景——东城风光正好，水波如绉纱般轻柔，迎接游船。绿杨如烟，晓寒轻淡，红杏枝头热闹非凡，"闹"字将静景化为动景，赋予了春天蓬勃的生命力，成为千古名句。宋祁因此得名"红杏尚书"。下片由景入情：人生苦短，欢娱太少，岂能吝惜千金而轻视那嫣然一笑？最后以拟人手法挽留夕阳："为君持酒劝斜阳，且向花间留晚照"，将对春光的依恋和珍惜之情写得极为浓丽缠绵。
