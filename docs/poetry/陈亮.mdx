# 陈亮

## 词人简介
陈亮（1143—1194），字同甫，号龙川，南宋著名思想家、文学家。力主抗金北伐，多次上书朝廷论恢复大计，终生未仕。其词风格豪放激昂，与辛弃疾交厚，有《龙川词》传世。

## 水龙吟·闹花深处层楼

**原文：**
> 闹花深处层楼，画帘半卷东风软。春归翠陌，平莎茸嫩，垂杨金浅。迟日催花，淡云阁雨，轻寒轻暖。恨芳菲世界，游人未赏，都付与、莺和燕。
>
> 寂寞凭高念远。向南楼、一声归雁。金钗斗草，青丝勒马，风流云散。罗绶分香，翠绡封泪，几多幽怨。正销魂，又是疏烟淡月，子规声断。

**创作背景：**
此词写春日登楼怀远之情。词人凭高远眺，面对无限春光却无人共赏，追念往昔与故人的欢聚时光，而今风流云散，只余幽怨与销魂。

**词意简析：**
上片写春日楼台景色：花深处层楼高耸，画帘半卷，东风轻软。春归大地，翠色满陌，平莎嫩绿，垂杨泛着浅浅的金色。迟日催花开，淡云敛雨，正是轻寒轻暖的宜人天气。然而如此芳菲世界，游人不来欣赏，都让莺燕独享——一个"恨"字点出词人心中寂寞。下片写登高念远的寂寥情怀。南楼之上听一声归雁，想起昔日金钗斗草、青丝勒马的欢聚时光，如今却已风流云散。罗绶分香、翠绡封泪的往事只剩几多幽怨。正在销魂之际，又是疏烟淡月，子规声声凄断，将愁情推向高潮。
