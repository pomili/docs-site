# 严仁

## 词人简介

严仁（生卒年不详），字次山，号樵溪，邵武（今福建邵武）人。南宋词人，与严羽、严参并称"邵武三严"。其词多写男女相思、羁旅离愁，风格婉约清丽，语言工致精巧。著有《清江欸乃集》，已佚，存词仅数首。

## 木兰花·春风只在园西畔

**原文：**
> 春风只在园西畔，荠菜花繁胡蝶乱。冰池晴绿照还空，香径落红吹已断。
>
> 意长翻恨游丝短，尽日相思罗带缓。宝奁明月不欺人，明日归来君试看。

**创作背景：**
此词写一位女子在春日思归的离愁别恨。南宋时期，文人或因科考、或因宦游常年在外，闺中思妇的相思之苦成为词中常见主题。严仁以工巧之笔写出了女子中心绪的千回百转。

**词意简析：**
上片写园中春景：春风只在一角吹拂，荠菜花繁、蝴蝶纷飞，池水澄碧映空，花径落红飘尽——写景中暗含孤寂，春色虽美却无人共赏。下片写相思之苦："意长翻恨游丝短"极见匠心——情意太长，反倒怨恨连游丝那样连接两地的东西都太短了；"尽日相思罗带缓"以衣带渐宽写消瘦，暗用古诗意。末二句"宝奁明月不欺人，明日归来君试看"——镜子和明月不会骗人，等你回来看看我憔悴了多少。全词构思巧妙，语言婉丽，是南宋闺情词中的佳作。
