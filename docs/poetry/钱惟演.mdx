# 钱惟演

## 词人简介
钱惟演（962—1034，一说977—1034），字希圣，临安（今浙江杭州）人，吴越王钱俶之子，随父降宋。官至枢密使、同中书门下平章事，后坐事贬谪汉东（随州）。他是"西昆体"代表诗人之一，与杨亿、刘筠等唱和，编有《西昆酬唱集》。其词以婉约见长，风格清丽哀婉。

## 木兰花·城上风光莺语乱

**原文：**
> 城上风光莺语乱，城下烟波春拍岸。绿杨芳草几时休？泪眼愁肠先已断。
>
> 情怀渐觉成衰晚，鸾镜朱颜惊暗换。昔年多病厌芳尊，今日芳尊惟恐浅。

**创作背景：**
此词作于词人晚年贬谪汉东（随州）期间。钱惟演原为吴越王子，降宋后虽官至高品，然晚年被贬，政治失意，处境凄凉。借伤春抒发迟暮之悲与身世之感，情调极为凄婉。

**词意简析：**
上片以明丽春景反衬内心凄黯——城上莺啼声声、城下春波拍岸，本是生机盎然的景象，然而词人却问"绿杨芳草几时休"，以"泪眼愁肠先已断"直抒悲痛。下片直写衰老之叹：情怀渐衰，镜中朱颜暗换，令人心惊。末二句"昔年多病厌芳尊，今日芳尊惟恐浅"是全词最精妙之笔——以借酒浇愁的行为对比，传神地写出今昔心境的巨大反差，从厌酒到唯恐酒杯太浅，绝望之情溢于言表。
