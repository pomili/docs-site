# 潘牥

## 词人简介

潘牥（1204—1246），字庭坚，号紫岩，福州闽县（今福建福州）人。南宋端平二年（1235年）进士，历任太学正、通判潭州等职。其词清丽沉郁，既有花间余韵，又含家国之忧。英年早逝，卒年四十三岁，存词不多。著有《紫岩集》。

## 南乡子·题南剑州妓馆

**原文：**
> 生怕倚阑干，阁下溪声阁外山。惟有旧时山共水，依然，暮雨朝云去不还。
>
> 应是蹑飞鸾，月下时时整佩环。月又渐低霜又下，更阑，折得梅花独自看。

**创作背景：**
此词题于南剑州（今福建南平）一家妓馆。词人旧地重游，伊人已去，唯有山水依旧。全词在凄清之景中寄托了对一位已逝或已离去的歌妓的深深怀念。

**词意简析：**
上片以"生怕倚阑干"开篇，一个"怕"字便奠定了全词的情感基调——因为倚栏所见之景会勾起对往事的回忆；"阁下溪声阁外山"写景如画，山水之景中最刺心的是"惟有旧时山共水，依然"——山水依旧，而"暮雨朝云去不还"，那个如暮雨朝云般的人却再也回不来了。下片以虚幻之笔写思念：料想她应是乘着飞鸾仙去了，月下时时整理佩环（将逝去的女子比作缥缈仙姝）；"月又渐低霜又下"写长夜将尽，寒霜又降；末句"折得梅花独自看"是全词最动人的画面——折下一枝梅花，独自在寒夜中凝视。这个孤独的姿势胜过千言万语，将无可言说的思念凝聚在了一个简静的画面中。全词短短数句，却写尽了物是人非的苍凉与思念之情的执着。
