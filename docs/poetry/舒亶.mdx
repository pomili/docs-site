# 舒亶

## 词人简介
舒亶（1041—1103），字信道，号懒堂，明州慈溪（今浙江宁波）人。北宋治平二年（1065年）进士，累官至御史中丞。曾参与弹劾苏轼，制造"乌台诗案"。后因与尚书省意见相左被逐出京城，晚年复起。其词以小令见长，风格清丽委婉，善于借景抒情，在北宋词坛自成一家。

## 虞美人·芙蓉落尽天涵水

**原文：**
> 芙蓉落尽天涵水，日暮沧波起。背飞双燕贴云寒，独向小楼东畔倚阑看。
>
> 浮生只合尊前老，雪满长安道。故人早晚上高台，寄我江南春色一枝梅。

**创作背景：**
此词为舒亶寄赠友人黄公度之作。元丰六年（1083），舒亶因与尚书省意见相左被逐出京城，在家赋闲十年后再度入京，然而物是人非，心生孤独与凄凉，渴望得到友人的音信，于是写下这首词寄给友人。化用陆凯折梅寄友的典故，表达深厚情谊与对温暖希望的期盼。

**词意简析：**
上片以秋景起笔——荷花落尽，水天相接一片苍茫，日暮时沧波涌起，一对燕子背向而飞，贴着寒云远去。词人独自倚在小楼东畔栏杆上凝望，景象苍茫萧索，烘托出孤寂惆怅的心境。下片直抒胸臆——浮生短暂，只应在酒杯前老去，转眼间大雪又落满了长安的街道。"浮生只合尊前老，雪满长安道"是全词的警句，以雪满长安渲染冷寂气氛，借酒遣日更显出孤凄无依的心境。末句从对方落笔，想象友人也每日早晚登台眺望，请寄我一枝江南报春的梅花。化用陆凯赠范晔的典故，将彼此的情谊与对春天的期盼融为一体，曲折委婉，意味深长。
