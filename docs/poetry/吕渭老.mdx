# 吕渭老

## 词人简介
吕渭老，字圣求，生卒年不详，北宋末南宋初词人。宣和、靖康年间在世，有《圣求词》传世。其词风格婉丽深婉，多写离愁别恨、男女相思之作。

## 薄幸·青楼春晚

**原文：**
> 青楼春晚。昼寂寂、梳匀又懒。乍听得、鸦啼莺哢，惹起新愁无限。记年时、偷掷春心，花间隔雾遥相见。便角枕题诗，宝钗贳酒，共醉青苔深院。
>
> 怎忘得、回廊下，携手处、花明月满。如今但暮雨，蜂愁蝶恨，小窗闲对芭蕉展。却谁拘管。尽无言、闲品秦筝，泪满参差雁。腰支渐小，心与杨花共远。

**创作背景：**
此词写一位女子在暮春时节的孤寂与愁思。词人细腻刻画了女子对往昔欢愉时光的追忆，以及如今独守空闺的凄凉。

**词意简析：**
上片起笔写暮春时节青楼寂寥，女子懒于梳妆，鸦啼莺啭反而勾起无限新愁。随即转入回忆，当年花间隔雾遥见心上人，偷掷春心暗许情意，而后角枕题诗、宝钗换酒，共醉于青苔深院之中，是何等甜蜜。下片回到现实，"怎忘得"三字引出对回廊携手、花明月满的深深怀念。而今只有暮雨纷纷，蜂愁蝶恨，独对小窗芭蕉。结以泪满秦筝、腰肢渐瘦，一颗心随杨花飘向远方，写出相思之苦的极致。
