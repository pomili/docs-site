# 范仲淹

## 词人简介
范仲淹（989—1052），字希文，谥号文正，苏州吴县人。北宋著名政治家、文学家，"庆历新政"主持者。其词作苍凉悲壮，开豪放派先声，以边塞题材和家国情怀著称，现存词仅五首，却篇篇精绝。

## 苏幕遮·碧云天

**原文：**
> 碧云天，黄叶地，秋色连波，波上寒烟翠。山映斜阳天接水，芳草无情，更在斜阳外。
> 黯乡魂，追旅思，夜夜除非，好梦留人睡。明月楼高休独倚，酒入愁肠，化作相思泪。

**创作背景：**
此词写于范仲淹任职西北边塞期间。作者长年戍守边疆，见秋景萧瑟，触发了浓烈的思乡之情，于是写下这首千古名篇。

**词意简析：**
上片以碧云、黄叶、寒烟、斜阳勾勒出一幅辽阔苍凉的秋景图，由近及远，由实入虚。下片直抒羁旅思乡之情，借酒浇愁却愁更愁，最终化作相思之泪。全词情景交融，境界宏大而不失深婉，是范仲淹边塞词的代表作。

## 御街行·纷纷坠叶飘香砌

**原文：**
> 纷纷坠叶飘香砌。夜寂静，寒声碎。真珠帘卷玉楼空，天淡银河垂地。年年今夜，月华如练，长是人千里。
> 愁肠已断无由醉，酒未到，先成泪。残灯明灭枕头欹，谙尽孤眠滋味。都来此事，眉间心上，无计相回避。

**创作背景：**
此词作于范仲淹出任陕西经略副使期间，镇守西北边境，与家人长期分离。秋夜独处，听到落叶声碎，望月怀人，写下了这首深沉的秋夜怀人之作。

**词意简析：**
上片从听觉入手，落叶坠阶的细碎声衬托秋夜的寂静，继而仰望银河垂地、月华如练的美景，却反衬出"人千里"的孤寂。下片直写愁绪之深——酒未饮已成泪，残灯孤枕，将离愁写得入骨三分。结尾"眉间心上"四字，凝练了愁绪由内而外无处不在的感受，成为千古传诵的名句。
