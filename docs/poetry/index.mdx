# 宋词三百首

《宋词三百首》由晚清词学大家**朱祖谋**（号上彊村民）于1924年编定，共收录两宋词人**84家**，词作**约300首**，是流传最广、影响最大的宋词选本。

---

## 凡例

每首词下附：
- **原文**：据通行版本校录
- **创作背景**：简述写作时间、地点与缘由
- **词意简析**：疏通大意，点明艺术特色
