# 毛滂

## 词人简介
毛滂（约1055—约1120），字泽民，号东堂，衢州江山（今属浙江）人。元祐中为杭州法曹，受知于苏轼。后任武康知县、秀州知州等职。有《东堂词》传世。其词风格婉丽缠绵，前人评其"语尽而意不尽，意尽而情不尽，何酷似秦少游也"。代表作《惜分飞》曾得苏轼击节赞赏。

## 惜分飞·泪湿阑干花著露

**原文：**
> 泪湿阑干花著露，愁到眉峰碧聚。此恨平分取，更无言语空相觑。
>
> 短雨残云无意绪，寂寞朝朝暮暮。今夜山深处，断魂分付潮回去。

**创作背景：**
此词为毛滂辞官离杭时，途经富阳、夜宿僧舍，为相爱的歌妓琼芳所作。据载，苏轼曾在席间听到歌妓唱此词，大为赞赏，称"郡僚有词人不及知，某之罪也"，可见此词在当时传唱之广。全词以女子口吻写离别之痛，将一个宦游公子与风尘女子之间真挚又无望的恋情写得凄婉动人。

**词意简析：**
上片以女子离别时的泪眼愁眉切入——泪水纵横的面庞如同沾满露珠的花朵，紧蹙的愁眉如碧山攒聚。"此恨平分取"一句将愁恨平分于两人，而"更无言语空相觑"写尽了离别时刻千言万语不知从何说起、只能默默对视的悲怆。下片以"短雨残云"暗用宋玉《高唐赋》的典故，既写眼前残败的景色，也暗示一段露水情缘的终结。最后以奇思作结——"今夜山深处，断魂分付潮回去"，将自己的一片痴魂托付给钱塘江潮，愿它随潮水回到情人身边。此句将刻骨的思念表达得极为奇警动人，是全词最精彩之笔。
