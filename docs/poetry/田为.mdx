# 田为

## 词人简介
田为，字不伐，生卒年不详，北宋末南宋初词人。善琵琶，通音律，曾任大晟府典乐。其词风格婉丽，存词仅六首。

## 江神子慢·玉台挂秋月

**原文：**
> 玉台挂秋月。铅素浅，梅花傅香雪。冰姿洁。金莲衬，小小凌波罗袜。雨初歇。楼外孤鸿声渐远，远山外、行人音信绝。此恨对语犹难，那堪是、寄书频说。
>
> 教人红销翠减，觉衣宽金缕，都为轻别。太情切。消魂处、画角黄昏时节。声呜咽。落尽庭花春去也，银蟾迥、无情圆又缺。恨伊不似余香，惹鸳鸯结。

**创作背景：**
此词写闺中女子对远方行人的深切思念。以秋夜为背景，借月、梅、雪等意象营造清冷孤寂的氛围，抒写离愁别恨。

**词意简析：**
上片从秋月写起，以铅粉淡妆、梅花映雪的意象烘托女子冰清玉洁的形象，随后写雨后孤鸿远去、行人音信全无，引出离恨。下片直抒胸臆，写因离别而容颜憔悴、衣带渐宽，黄昏画角声中更添凄凉。结尾以月圆月缺的无情反衬自己的深情，以鸳鸯结的意象寄托对团聚的渴望。全词情致缠绵，婉约动人。
