# 张元幹

## 词人简介
张元幹（1091—约1170），字仲宗，号芦川居士，永福（今福建永泰）人。南宋初年爱国词人，曾为李纲幕僚，力主抗金，后因作词送胡铨被除名。其词慷慨悲凉，豪放中寓沉郁，对南宋豪放派词风影响深远。

## 石州慢·寒水依痕

**原文：**
> 寒水依痕，春意渐回，沙际烟阔。溪梅晴照生香，冷蕊数枝争发。天涯旧恨，试看几许消魂？长亭门外山重叠。不尽眼中青，是愁来时节。
>
> 情切。画楼深闭，想见东风，暗消肌雪。孤负枕前云雨，尊前花月。心期切处，更有多少凄凉，殷勤留与归时说。到得再相逢，恰经年离别。

**创作背景：**
此词写于北宋末南宋初，张元幹因主张抗金遭贬谪流落江南之时。词人怀乡念远，借写早春景色，抒发羁旅漂泊之愁和对故国家山的深切思念。

**词意简析：**
上片写早春景象，寒水初涨，溪梅争发，然而词人面对远山长亭，只觉愁思无尽。"不尽眼中青，是愁来时节"将春色与愁绪融为一片。下片由室外转向室内，想象闺中人的孤寂，再转到自身的凄凉，结句"到得再相逢，恰经年离别"道尽久别重逢的感慨，沉痛深切。

## 兰陵王·卷珠箔

**原文：**
> 卷珠箔。朝雨轻阴乍阁。阑干外，烟柳弄晴，芳草侵阶映红药。东风妒花恶。吹落。梢头嫩萼。屏山掩，沉水倦熏，中酒心情怯杯勺。
>
> 寻思旧京洛。正年少疏狂，歌笑迷著。障泥油壁催梳掠。曾驰道同载，上林携手，灯夜初过早共约。又争信飘泊？
>
> 寂寞。念行乐。甚粉淡衣襟，音断弦索。琼枝璧月春如昨。怅别后华表，那回辽鹤。相思除是，向醉里、暂忘却。

**创作背景：**
此词为张元幹晚年追忆北宋汴京繁华之作。靖康之变后，词人流落江南，时常追怀当年在汴京的欢乐时光，以寄托故国之思。"兰陵王"为北宋流行的大曲调，声情激越。

**词意简析：**
全词分三叠。第一叠写眼前春景与酒醒后的惆怅，以"中酒心情怯杯勺"开篇；第二叠追忆汴京少年时光，"驰道同载，上林携手"描绘了当年繁华盛景，"又争信飘泊"一句陡然转折，点出如今流落之痛；第三叠抒写别后寂寞，"琼枝璧月春如昨"物是人非之感尤为深切，结句道出唯有醉酒方能暂忘相思之苦。
