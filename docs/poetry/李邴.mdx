# 李邴

## 词人简介
李邴（1085—1146），字汉老，号云龛，济州任城（今山东济宁）人。北宋末南宋初词人，官至参知政事。其词以咏物见长，意境清雅，托物寄怀，于婉约中含深沉。

---

## 汉宫春·潇洒江梅

**原文：**
> 潇洒江梅，向竹梢疏处，横两三枝。东君也不爱惜，雪压霜欺。无情燕子，怕春寒、轻失花期。却是有、年年塞雁，归来曾见开时。
>
> 清浅小溪如练，问玉堂何似，茅舍疏篱。伤心故人去后，冷落新诗。微云淡月，对江天、分付他谁。空自忆、清香未减，风流不在人知。

**创作背景：**
此词为咏梅之作，借梅花高洁孤芳的品格，抒发词人对故人的怀念以及自身洁身自好、不求人知的高尚情怀。当作于南渡之后，寄托了深沉的故国之思。

**词意简析：**
上片写江边梅花在竹梢疏处横斜开放的清雅姿态，春神不惜、雪压霜欺，燕子怕寒失约，唯有塞雁年年归来见证花开，以对比手法突出梅花的孤高与坚贞。下片转写清溪茅舍间的梅，以"玉堂"与"茅舍疏篱"对比，表明梅花不论在何处都自持其香。"伤心故人去后，冷落新诗"流露出知音难觅之痛，结句"清香未减，风流不在人知"既是写梅，亦是自况，表达不求闻达、自守高洁的志趣。
