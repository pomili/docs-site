# 僧挥

## 词人简介
僧挥，又作僧晖，字仲殊，北宋僧人。本姓张氏，名挥，安州（今湖北安陆）人。曾举进士，后弃家为僧，寓居苏州承天寺、杭州宝月寺。与苏轼交游甚密，其词清丽婉约，风格独特。

## 金明池·天阔云高

**原文：**
> 天阔云高，溪横水远，晚日寒生轻晕。闲阶静、杨花渐少，朱门掩、莺声犹嫩。悔匆匆、过却清明，旋占得余芳，已成幽恨。却几日阴沉，连宵慵困，起来韶华都尽。
>
> 怨入双眉闲斗损。乍品得情怀，看承全近。深深态、无非自许，厌厌意、终羞人问。争知道、梦里蓬莱，待忘了余香，时传音信。纵留得莺花，东风不住，也则眼前愁闷。

**创作背景：**
此词写暮春愁思，可能是僧挥寄托身世感慨之作。僧人写情词在宋代并不鲜见，往往寓含更深层的禅意或人生感悟。

**词意简析：**
上片以晚春景象铺写愁绪：天高云阔、溪水悠远，落日寒晕中杨花渐少、莺声犹嫩，一派暮春萧索。后悔匆匆过了清明，未能细细品味春光，而余芳已在几日阴沉中变成幽恨。下片转入内心独白：双眉愁损，情怀乍品，深深之态无须自许，厌厌之意羞于被人问及。梦里蓬莱（仙境/理想境界）虽留余香，却时传音信，不得长久。末以纵留莺花而东风不住、愁闷仍在前作结，写出愁绪之无法排遣。全词语言清丽，情感细腻。
