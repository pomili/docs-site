# 晁元礼

## 词人简介
晁元礼，字次膺，祖籍澶州清丰（今属河南），后徙居彭门（今江苏徐州）。北宋熙宁年间进士，曾任县令，后因事罢官，闲居乡里。其词多写风月恋情与闲居生活，风格婉约绮丽。

## 绿头鸭·咏月（晚云收）

**原文：**
> 晚云收，淡天一片琉璃。烂银盘、来从海底，皓色千里澄辉。莹无尘、素娥淡伫，静可数、丹桂参差。玉露初零，金风未凛，一年无似此佳时。露坐久，疏萤时度，乌鹊正南飞。瑶台冷，栏干凭暖，欲下迟迟。
>
> 念佳人、音尘别后，对此应解相思。最关情、漏声正永，暗断肠、花影偷移。料得来宵，清光未减，阴晴天气又争知。共凝恋、如今别后，还是隔年期。人强健，清尊素影，长愿相随。

**创作背景：**
此词为中秋咏月怀人之作。词人对月抒怀，既写月下美景，又寄离人之思，情感真挚而格调清雅。

**词意简析：**
上片写中秋月出之美：晚云散去，天空如琉璃般明净。皓月如银盘从海底升起，千里澄辉。月宫中素娥（嫦娥）淡伫无尘，丹桂参差可数。玉露初零、金风未凛，一年之中无如此佳时。露坐久之，流萤时飞、乌鹊南飞，栏干都凭暖了仍不忍离去。下片由月及人：念及佳人别后音尘相隔，对此明月应更解相思之苦。漏声正长、花影暗移，写夜深难寐。料想明宵清光犹在，但阴晴难料，不如珍惜今宵。末以祝福语作结，化用苏轼"但愿人长久，千里共婵娟"之意，愿清尊与明月长伴，人月两团圆。
