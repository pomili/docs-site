# 曹组

## 词人简介
曹组，字元宠，生卒年不详，北宋词人。宣和年间进士，官至阁门宣赞舍人。其词以俳谐著称，亦有清丽婉约之作，存词三十余首。

---

## 蓦山溪·梅

**原文：**
> 洗妆真态，不作铅花御。竹外一枝斜，想佳人，天寒日暮。黄昏院落，无处著清香，风细细，雪垂垂，何况江头路。
>
> 月边疏影，梦到消魂处。结子欲黄时，又须作，廉纤细雨。孤芳一世，供断有情愁，消瘦损，东阳也，试问花知否。

**创作背景：**
此词为咏梅之作，借梅花的高洁孤芳来寄托词人自身的身世感慨。梅花在宋词中常被赋予清高、孤傲的品格象征。

**词意简析：**
上片写梅花洗尽铅华、天然本色，以竹外一枝斜出的姿态出现在天寒日暮之中，虽有清香却无人欣赏。下片写月下梅影令人销魂，待到梅子将黄时又要经历细雨摧残。结尾以梅花孤芳一世，勾起有情人无限愁绪，反问花是否知晓自己的消瘦与苦闷。词人借梅自喻，表达怀才不遇的孤寂之情。
