# 蔡伸

## 词人简介
蔡伸（1088—1156），字伸道，号友古居士，莆田（今属福建）人。南宋词人，官至左中大夫。其词风婉约清丽，善于写景抒情，尤其擅长以秋景、雁声等意象表达羁旅愁怀与离别之情。

## 苏武慢·雁落平沙

**原文：**
> 雁落平沙，烟笼寒水，古垒鸣笳声断。青山隐隐，败叶萧萧，天际暝鸦零乱。楼上黄昏，片帆千里归程，年华将晚。望碧云空暮，佳人何处，梦魂俱远。
>
> 忆旧游、邃馆朱扉，小园香径，尚想桃花人面。书盈锦轴，恨满金徽，难写寸心幽怨。两地离愁，一尊芳酒凄凉，危阑倚遍。尽迟留、凭仗西风，吹干泪眼。

**创作背景：**
此词写于词人羁旅途中，登楼眺望秋景，触目伤怀而思佳人。蔡伸一生宦游四方，常有离愁别恨之作，此词即其羁旅怀人的代表篇章。

**词意简析：**
上片以六个四字句连缀出一幅苍凉秋景：平沙落雁、寒水笼烟、古垒鸣笳、青山败叶、暝鸦零乱，层层渲染出深秋薄暮的萧瑟氛围。"年华将晚"四字将自然之景与人生迟暮的感伤融为一体。下片由"忆旧游"领起对往昔美好时光的追忆，"桃花人面"用崔护诗意，暗示物是人非。"两地离愁，一尊芳酒凄凉"极写相思之苦，结尾"凭仗西风，吹干泪眼"情深而语绝，尤为动人。

## 柳梢青·数声鶗鴂

**原文：**
> 数声鶗鴂。可怜又是，春归时节。满院东风，海棠铺绣，梨花飘雪。
>
> 丁香露泣残枝，算未比、愁肠寸结。自是休文，多情多感，不干风月。

**创作背景：**
此词为暮春感怀之作。词人听到杜鹃啼鸣，意识到春已将尽，面对满院落花景象，不禁触动满怀愁绪。"休文"即南朝沈约，词人以多病多愁的沈约自比，表达心中郁结。

**词意简析：**
上片以杜鹃啼声起笔，直点春归。"海棠铺绣，梨花飘雪"八个字将落花景象写得极富美感，海棠落红如铺锦绣，梨花飞白若飘雪花，色彩对比鲜明。下片由落花写到丁香露泣残枝，以花喻人，"愁肠寸结"更进一层。结尾"自是休文，多情多感，不干风月"尤为警策——词人自认愁绪本因自身多情多感而起，并非外物所致，表面洒脱实则更深一层地写出愁之难解难排。
