# 王安国

## 词人简介
王安国（1028—1074），字平甫，抚州临川（今属江西）人，王安石之弟。北宋熙宁元年（1068）赐进士出身，历官大理寺丞、集贤校理等职。因受郑侠案牵连被放归田里。有《王校理集》传世，《全宋词》录其词三首。其词风清新婉丽，虽存词不多，但篇篇精妙，不依傍其兄王安石的门户而自成一家。

## 清平乐·留春不住

**原文：**
> 留春不住，费尽莺儿语。满地残红宫锦污，昨夜南园风雨。
>
> 小怜初上琵琶，晓来思绕天涯。不肯画堂朱户，春风自在杨花。

**创作背景：**
此词为伤春之作。王安国虽为宰相王安石之弟，却并不依附新党权势。词中以杨花自喻，表达了不亲近权贵、追求自由清高的品格。全词情景交融，在伤春的基调中寄寓了高洁的人格理想。

**词意简析：**
上片以倒装句式描绘暮春萧瑟——黄莺费尽啼声也留不住春天，昨夜南园风雨肆虐，满地的落花如被玷污了的宫锦一般残败不堪。下片先写歌女弹起琵琶，晓来思绪已飞绕天涯，接着笔锋一转，以杨花不肯飞入权贵人家的画堂朱户、只愿随春风自由飘舞作结。杨花既是暮春之景，也是词人自身品格的写照——不慕富贵，自在不拘。全词由伤春而及咏怀，层层转折，余韵悠长。
