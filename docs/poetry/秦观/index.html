秦观

词人简介​

秦观（1049—1100），字少游，又字太虚，号邗沟居士，世称淮海先生，高邮（今江苏高邮）人。北宋著名词人，"苏门四学士"之一，被尊为婉约派一代词宗。秦观一生坎坷，仕途多舛，因新旧党争牵连屡遭贬谪，最终病逝于赦还途中。其词以情韵取胜，风格婉约含蓄，意境深远凄美，语言清丽自然。冯煦评为"情韵兼胜"，王国维在《人间词话》中称其"词最凄婉"。有《淮海集》《淮海词》传世。

望海潮·梅英疏淡​

原文：

梅英疏淡，冰澌溶泄，东风暗换年华。金谷俊游，铜驼巷陌，新晴细履平沙。长记误随车。正絮翻蝶舞，芳思交加。柳下桃蹊，乱分春色到人家。

西园夜饮鸣笳。有华灯碍月，飞盖妨花。兰苑未空，行人渐老，重来是事堪嗟。烟暝酒旗斜。但倚楼极目，时见栖鸦。无奈归心，暗随流水到天涯。

创作背景： 此词约作于宋哲宗绍圣元年（1094年）春，新旧党争再起，秦观被贬出京前夕。词人重游洛阳，缅怀昔日与友人在此游乐的盛况，面对政治风波与个人贬谪的命运，感怀今昔而作。词中"金谷""铜驼"皆为洛阳名胜，故又题作《望海潮·洛阳怀古》。

词意简析： 上片追忆昔日洛阳春日游乐之盛。以梅花稀疏、冰澌消融写早春节候，"东风暗换年华"既写季节更替，也暗喻政局变化。金谷俊游、铜驼巷陌，记与友人散步赏春之乐；柳下桃蹊，春色满城，一派繁华。下片转入今昔之慨——西园夜饮、华灯飞盖的盛况犹在眼前，而如今"兰苑未空，行人渐老"，重游旧地已是人事全非。烟暝酒旗斜、时见栖鸦，以萧瑟之景写落寞之情。末句"无奈归心，暗随流水到天涯"收束全篇，将归思与流水融为一体，含蓄深远。

八六子·倚危亭​

原文：

倚危亭，恨如芳草，萋萋刬尽还生。念柳外青骢别后，水边红袂分时，怆然暗惊。

无端天与娉婷，夜月一帘幽梦，春风十里柔情。怎奈向、欢娱渐随流水，素弦声断，翠绡香减，那堪片片飞花弄晚，蒙蒙残雨笼晴。正销凝，黄鹂又啼数声。

创作背景： 此词是秦观怀旧伤别之作，约写于词人中年以后。秦观一生情感丰富而细腻，其词中多追忆往昔恋情，此词以芳草喻离恨，以飞花残雨写青春逝去的感伤，是淮海词中最见深情的作品之一。

词意简析： 上片以"倚危亭"起笔，"恨如芳草，萋萋刬尽还生"以春草除而还生的特性比喻离愁挥之不去。念及柳外青骢离别、水边红袂分手之时，仍然暗自惊心。下片追忆昔日佳人——天赐她那般美好，夜月一帘、春风十里，何等柔情缱绻。然而欢娱如流水般逝去，素弦声断、翠绡香减。结尾"片片飞花弄晚，蒙蒙残雨笼晴"以暮春花雨写美好时光的消逝，"正销凝，黄鹂又啼数声"——正当黯然出神之际，又听黄鹂数声啼叫，以动衬静，将愁情推至极致。

满庭芳·山抹微云​

原文：

山抹微云，天连衰草，画角声断谯门。暂停征棹，聊共引离尊。多少蓬莱旧事，空回首、烟霭纷纷。斜阳外，寒鸦万点，流水绕孤村。

销魂当此际，香囊暗解，罗带轻分。谩赢得、青楼薄幸名存。此去何时见也？襟袖上、空惹啼痕。伤情处，高城望断，灯火已黄昏。

创作背景： 此词是秦观最负盛名的词作之一，大约写于元丰年间，词人在会稽（今浙江绍兴）与一位歌妓离别时所作。因开篇"山抹微云"四字精妙绝伦，苏轼戏称秦观为"山抹微云秦学士"。

词意简析： 上片写别时之景与别后之思。"山抹微云，天连衰草"以"抹""连"二字写天际远景，画角声断，征棹将发。蓬莱旧事（指在会稽的欢愉时光）已如烟霭般缥缈难追。"斜阳外，寒鸦万点，流水绕孤村"以三组意象叠合，构成一幅苍茫凄凉的别后图景。下片写离别之痛——香囊暗解、罗带轻分，是离别时互赠信物的细节。"漫赢得青楼薄幸名存"化用杜牧"十年一觉扬州梦，赢得青楼薄幸名"诗意，自嘲之中饱含无奈。末句"高城望断，灯火已黄昏"以景结情，将无尽的离愁融入苍茫暮色之中。此词一出，轰动一时，是婉约词的经典之作。

满庭芳·晓色云开​

原文：

晓色云开，春随人意，骤雨才过还晴。古台芳榭，飞燕蹴红英。舞困榆钱自落，秋千外、绿水桥平。东风里，朱门映柳，低按小秦筝。

多情，行乐处，珠钿翠盖，玉辔红缨。渐酒空金榼，花困蓬瀛。豆蔻梢头旧恨，十年梦、屈指堪惊。凭阑久，疏烟淡日，寂寞下芜城。

创作背景： 此词作于扬州，是一首伤春怀旧之作。秦观曾任职扬州，对这座古城有着深厚的感情。词人在某个春日早晨，面对雨过天晴的美景，不由回忆起往昔在扬州的行乐时光，感叹十年如梦。

词意简析： 上片写春日清晨雨过天晴的明媚景色——晓色云开，骤雨还晴，古台芳榭之间飞燕蹴花，秋千外绿水与桥齐平。东风里，朱门掩映在柳丝中，隐约传来秦筝的低按声。画面清新明丽，充满生机。下片转入怀旧——当年扬州行乐处，珠钿翠盖、玉辔红缨，何等热闹奢华。然而酒尽花困，豆蔻年华的旧恨，十年过去屈指堪惊。末三句"凭阑久，疏烟淡日，寂寞下芜城"以寂寞之境收束全词，与上片的明媚春光形成强烈对比，写出繁华落尽后的沧桑感。

减字木兰花·天涯旧恨​

原文：

天涯旧恨，独自凄凉人不问。欲见回肠，断尽金炉小篆香。

黛蛾长敛，任是春风吹不展。困倚危楼，过尽飞鸿字字愁。

创作背景： 此词写闺中女子对远方情人的离愁别恨，具体年代不详。秦观善于代言体，能以女子口吻写出细腻深婉的情感。词中"困倚危楼，过尽飞鸿字字愁"是千古传诵的名句。

词意简析： 上片写独处之凄苦——天涯路远，旧恨难消，独自凄凉无人过问。将愁肠比作金炉中燃尽的小篆香，香烟缭绕而终至断灭，形象地写出愁思之深与绝望之至。下片写愁眉不展——黛眉长敛，任凭春风吹拂也无法舒展。困倚危楼，望着天空中南归的飞鸿，它们或排人字、或排一字，每一个字看在眼里都变成了"愁"字。"过尽飞鸿字字愁"一语双关，既是写雁阵之形，也是写心中之愁，堪称绝妙。

浣溪沙·漠漠轻寒上小楼​

原文：

漠漠轻寒上小楼，晓阴无赖似穷秋。淡烟流水画屏幽。

自在飞花轻似梦，无边丝雨细如愁。宝帘闲挂小银钩。

创作背景： 此词为秦观小令中的压卷之作，具体年代不详。词中写一个女子在春阴清晨中生发的淡淡哀愁与寂寞，被前人评为"奇语"。全词不着一个"愁"字，而愁情无处不在。

词意简析： 上片写环境——漠漠轻寒，小楼幽寂，拂晓的阴沉竟如深秋一般令人无奈。室内画屏上淡烟流水，意境清幽。下片写窗外之景与内心之情——"自在飞花轻似梦，无边丝雨细如愁"是千古名句。飘飞的花瓣轻盈如梦，漫天的细雨纤细如愁，将无形之情化为可感的意象，比喻新颖贴切。"宝帘闲挂小银钩"以帘卷闲置的悠闲之态收束，看似平淡，实则暗写无心赏景的落寞。全词以轻浅的色调写深微的愁绪，婉约轻柔，意蕴悠长，代表着秦观词柔婉精微的最高成就。

阮郎归·湘天风雨破寒初​

原文：

湘天风雨破寒初，深沉庭院虚。丽谯吹罢《小单于》，迢迢清夜徂。

乡梦断，旅魂孤。峥嵘岁又除。衡阳犹有雁传书，郴阳和雁无。

创作背景： 此词作于宋哲宗绍圣三年（1096年）除夕，秦观被贬谪至郴州（今湖南郴州）时所作。这是秦观贬谪生涯中最凄凉的时期，远离家乡，音信隔绝，又逢除夕岁除，孤独悲苦之情达于极致。此词被公认为淮海词中情调最为凄婉的作品。

词意简析： 上片写贬所的荒寒——湘天的风雨吹破了初寒，深深的庭院空洞虚寂。城楼上吹罢《小单于》的乐曲，漫漫清冷的除夕之夜在寂寥中一点点流逝。下片写思乡之苦——乡梦难成，旅魂孤苦，又到了一年将尽的除夕。末二句是全词的警句：衡阳还有大雁可以传书，郴阳在衡阳之南，连大雁也飞不到，更不用说书信往来了。"衡阳犹有雁传书，郴阳和雁无"以层递手法将绝望推至极致——别人尚且可以收到雁书，而我连大雁都没有了。此句与晏几道"梦魂纵有也成虚，那堪和梦无"并称双璧，同为词史上最凄美的句子之一。
