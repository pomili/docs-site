# 张镃

## 词人简介

张镃（1153—1221），字功甫，号约斋，南宋著名词人、诗人。出身贵胄，是南宋名将张俊的曾孙，家财丰饶。其词风清丽幽雅，以咏物见长，与姜夔、杨万里等交游甚密。晚年因卷入朝廷斗争遭贬，死于象州。著有《玉照堂词》（又名《南湖诗余》）。

## 满庭芳·促织儿

**原文：**
> 月洗高梧，露漙幽草，宝钗楼外秋深。土花沿翠，萤火坠墙阴。静听寒声断续，微韵转、凄咽悲沉。争求侣，殷勤劝织，促破晓机心。
>
> 儿时，曾记得，呼灯灌穴，敛步随音。任满身花影，犹自追寻。携向华堂戏斗，亭台小、笼巧妆金。今休说，从渠床下，凉夜伴孤吟。

**创作背景：**
此词与姜夔《齐天乐·蟋蟀》同为咏蟋蟀之作。据姜夔词序，庆元二年（1196年）秋，张镃与姜夔等友人在张府宴饮，闻蟋蟀之声，相约同赋蟋蟀。张镃先成此词，姜夔继之。两首咏蟋蟀词各擅胜场，成为词坛佳话。

**词意简析：**
上片写秋夜听蟋蟀之声：月光洗桐、露湿幽草，秋意已深；萤火坠墙、土花沿翠，是蟋蟀出没的环境；"静听寒声断续，微韵转凄咽悲沉"以极为细腻的笔触写出蟋蟀鸣声的断续、悲戚与低沉。"争求侣"点出蟋蟀鸣叫求偶的本能，"促破晓机心"化用"促织"之名——蟋蟀又叫促织，仿佛催促织女赶工。下片追忆儿时捉蟋蟀的欢乐：呼灯灌穴、敛步随音，写得生动活泼；"任满身花影犹自追寻"写儿时痴迷之态。末三句笔锋一转回到当下——如今老了，再也没有那番兴致，任凭蟋蟀在床下伴着寒夜孤吟。全词以儿时之乐衬今日之孤，在咏物中寄托了深沉的人生感慨。

## 宴山亭·幽梦初回

**原文：**
> 幽梦初回，重阴未开，晓色催成疏雨。竹槛气寒，蕙畹声摇，新绿暗通南浦。未有人行，才半启、回廊朱户。无绪，空望极霓旌，锦书难据。
>
> 苔径追忆曾游，念谁伴秋千，彩绳芳柱。犀帘黛卷，凤枕云孤，应也几番凝伫。想得伊家，也应是、眼波横注。归来否，但梦里、分明相遇。

**创作背景：**
此词写春日怀人。词人借梦境与现实交织的手法，表达对远方恋人的深切思念。全词以景写情，意象幽雅，情感含蓄而深挚。

**词意简析：**
上片写梦醒后的清晨：幽梦初回，天色阴沉的早晨，稀疏的雨声催人清醒；竹槛寒凉，香草在风中摇曳，新绿蔓延到南浦（送别之地）。"空望极霓旌，锦书难据"——遥望天边，希望看到使者的霓旌带来书信，却空等一场。下片追忆旧游："苔径追忆曾游"写旧地重游，"念谁伴秋千，彩绳芳柱"想象伊人荡秋千的身影；"犀帘黛卷，凤枕云孤"以冷寂的居室写孤独的思念。末四句是全词最动人的地方："想得伊家也应是眼波横注"——料想你也在那里凝望归路，眼波含情；"归来否，但梦里分明相遇"——到底回不回来呢？也只有在梦中与你分明相遇了。全词将梦与现实、实写与虚写交织一处，情感层层递进，是南宋怀人词中的佳作。
