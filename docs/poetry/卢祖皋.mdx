# 卢祖皋

## 词人简介

卢祖皋（约1174—1224），字申之，又字次夔，号蒲江，温州永嘉（今浙江温州）人。南宋庆元五年（1199年）进士，官至权直学士院。其词学周邦彦，风格婉约清丽，在浙东词派中自成一家。其小令意境深远，琢语工巧，颇为时人称道。著有《蒲江词稿》。

## 江城子·画楼帘幕卷新晴

**原文：**
> 画楼帘幕卷新晴。掩银屏，晓寒轻。坠粉飘香，日日唤愁生。暗数十年湖上路，能几度，著娉婷。
>
> 年华空自感飘零。拥春酲，对谁醒。天阔云闲，无处觅箫声。载酒买花年少事，浑不似，旧心情。

**创作背景：**
此词写春日伤怀。词人回首十年湖上生涯，感年华之飘零，叹心情之不再，将对往昔的怀念与对当下的怅惘交织一处，写尽中年文人对时光流逝的无奈。

**词意简析：**
上片写春晨景象：画楼帘幕卷起，新晴乍现，晓寒犹轻；落花飘香，日日勾起愁绪；"暗数十年湖上路，能几度著娉婷"——默默回想十年来的漂泊，有几次能与佳人相伴？一个"暗"字写出心事之隐秘，一个"几度"写尽聚少离多的遗憾。下片抒情："年华空自感飘零"是对人生漂泊的深刻感悟；"天阔云闲，无处觅箫声"化用杜牧诗意——天地虽广阔，却再也找不到当年那动人的箫声，暗喻美好时光不可重寻。末句"载酒买花年少事，浑不似旧心情"——载酒买花是少年时常做的事，如今再做，心境已完全不同。此词语言清丽，情感深沉，是卢祖皋小令中的佳作。

## 宴清都·春讯飞琼管

**原文：**
> 春讯飞琼管，风日薄、度墙啼鸟声乱。江城次第，笙歌翠合，绮罗香暖。溶溶涧渌冰泮，醉梦里、年华暗换。料黛眉、重锁隋堤，芳心还动梁苑。
>
> 新来雁阔云音，鸾分鉴影，无计重见。啼春细雨，笼愁澹月，恁时庭院。离肠未语先断，算犹有、凭高望眼。更那堪、芳草连天，飞梅弄晚。

**创作背景：**
此词写春日怀人。词人以春日万象更新之景衬托自己别离的孤寂，在绮丽的春色中寄寓深深的思念之情。全词以景起兴，以情收束，是卢祖皋婉约词的代表作。

**词意简析：**
上片写春回大地的景象：春的消息从律管中飞出（古人以葭灰测节气），风和日暖，鸟声穿墙；整座江城渐次繁华，笙歌四起，仕女如云；溪水溶溶，冰消雪化，在醉梦中不知不觉年华暗换。"料黛眉重锁隋堤，芳心还动梁苑"以拟人手法写杨柳（黛眉）新绿和花草（芳心）萌动。下片转入怀人：近来鸿雁无信、照镜自怜（鸾分鉴影用鸾鸟见镜中孤影而悲鸣之典），与伊人无计重见；"啼春细雨，笼愁澹月"——细雨如啼、淡月笼愁，院中最易触动离肠；"离肠未语先断"写相思之深切入骨。末句"更那堪芳草连天，飞梅弄晚"——放眼芳草连天，梅花飘落在暮色中，以辽阔之景写无尽之愁。全词意象绮丽而不繁缛，情感深婉而不纤弱，是南宋婉约词中咏怀词的佳作。
