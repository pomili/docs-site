# 陆淞

## 词人简介
陆淞，字子逸，号云溪，生卒年不详，南宋词人。陆游的长兄，曾任辰州通判。其词风格婉约清丽，传世作品仅两首。

## 瑞鹤仙·脸霞红印枕

**原文：**
> 脸霞红印枕。睡觉来、冠儿还是不整。屏间麝煤冷。但眉峰压翠，泪珠弹粉。堂深昼永。燕交飞、风帘露井。恨无人，与说相思，近日带围宽尽。
>
> 重省。残灯朱幌，淡月纱窗，那时风景。阳台路迥。云雨梦，便无准。待归来，先指花梢教看，欲把心期细问。问因循、过了青春，怎生意稳。

**创作背景：**
此词写一位女子午睡醒来后的相思之苦。词人细腻描绘女子的神态与心理活动，将她对情人的思念与幽怨刻画得淋漓尽致。

**词意简析：**
上片写女子午睡醒来，脸上还印着枕痕，冠儿不整，慵懒无心梳妆。屏风间麝煤已冷，眉峰紧蹙、泪珠暗弹。深堂之中白日漫长，燕子双双穿帘而飞，更衬出她的孤单。没有人可诉说相思之苦，近日来腰带已宽尽，瘦损不堪。下片转入回忆，残灯朱幌、淡月纱窗，那时风光历历在目。如今阳台路远，云雨之梦无准无凭。等到他归来，要指着花梢让他看看，想问问他：为何因循拖延空过了青春，你怎能心安理得？结尾将女子的幽怨与质问表达得入木三分。
