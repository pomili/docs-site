# 潘希白

## 词人简介
潘希白，字怀古，号渔庄，永嘉（今浙江温州）人，南宋宝祐年间进士。曾任干办临安府节制司公事，宋亡后隐居不仕。其词感慨深沉，多故国之思。

---

## 大有·九日

**原文：**
> 戏马台前，采花篱下，问岁华、还是重九。恰归来、南山翠色依旧。帘栊昨夜听风雨，都不似、登临时候。一片宋玉情怀，十分卫郎清瘦。
>
> 红萸佩，空对酒。砧杵动微寒，暗欺罗袖。秋已无多，早是败荷衰柳。强整帽檐欹侧，曾经向、天涯搔首。几回忆、故国莼鲈，霜前雁后。

**创作背景：**
此词作于重阳佳节，词人以登高怀远之传统起兴，抒发身世飘零、故国之思的深沉感慨。

**词意简析：**
上片以重阳登高的典故开篇，引陶渊明采菊东篱、宋玉悲秋、卫玠清瘦之典，自比清瘦多愁。风雨之后南山翠色依旧，而人事已非，愈显沧桑。下片以茱萸、清酒、砧杵等重阳意象层层推进，败荷衰柳暗示时序将尽、国运衰微。末句化用张翰莼鲈之思的典故，表达对故国的深切怀念。全词含蓄蕴藉，寄托遥深。
