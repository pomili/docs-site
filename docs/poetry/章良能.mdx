# 章良能

## 词人简介

章良能（？—1214），字达之，处州丽水（今浙江丽水）人。南宋淳熙五年（1178年）进士，官至参知政事（副宰相）。其词仅存一首，收入《绝妙好词》。虽只一首，却以清丽婉约、含蓄蕴藉著称。

---

## 小重山·柳暗花明春事深

**原文：**
> 柳暗花明春事深。小阑红芍药，已抽簪。雨余风软碎鸣禽。迟迟日，犹带一分阴。
>
> 往事莫沉吟。身闲时序好，且登临。旧游无处不堪寻。无寻处，惟有少年心。

**创作背景：**
此词写于词人晚年，时在南宋偏安时期。章良能位居高位，政务之余感怀往事，借春景抒发对少年时光的追忆。春日登临，满目花明柳暗，处处可寻旧游之地，唯独少年心境再难找回。

**词意简析：**
上片写暮春景色：柳暗花明，红芍药抽芽，雨后风轻鸟鸣，春日迟迟——一幅生机盎然的春景图。下片由景生情，劝自己"往事莫沉吟"，趁着清闲去登临游玩；"旧游无处不堪寻，无寻处，惟有少年心"是全词点睛之笔——旧地可以重游，旧事可以追忆，但少年时那份纯真无忧的心境却再也找不回来了。这两句以极朴素的语言道出了所有人都能感同身受的人生遗憾。
