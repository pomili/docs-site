# 蒋捷

## 词人简介
蒋捷，字胜欲，号竹山，阳羡（今江苏宜兴）人，宋度宗咸淳年间进士。宋亡后隐居太湖竹山，终身不仕。其词多写亡国之痛、身世之感，风格悲凉清俊，语言洗练，与周密、王沂孙、张炎并称"宋末四大家"。有《竹山词》传世。

## 贺新郎·梦冷黄金屋

**原文：**
> 梦冷黄金屋。叹秦筝、斜鸿阵里，素弦尘扑。化作娇莺飞归去，犹认纱窗旧绿。正过雨、荆桃如菽。此恨难平君知否，似琼台、涌起弹棋局。消瘦影，嫌明烛。
>
> 鸳楼碎泻东西玉。问芳踪、何时再展，翠钗难卜。待把宫眉横云样，描上生绡画幅。怕不是、新来妆束。彩扇红牙今都在，恨无人、解听开元曲。空掩袖，倚寒竹。

**创作背景：**
此词约作于宋亡之后，词人借梦境追怀故国，以昔日繁华与今日凄凉对照，抒写亡国之痛和身世飘零之感。

**词意简析：**
上片以"梦冷黄金屋"起句，黄金屋为汉武帝金屋藏娇之典，此处暗喻故宫。梦中昔日繁华已冷，秦筝尘封，唯有化作娇莺归去还能认得旧时纱窗绿色。过雨之后樱桃如豆，季节转换中隐含沧桑。此恨难平，如琼台弹棋之局纷乱难解，消瘦身影怕见明烛，写尽孤寂。下片以鸳楼玉碎喻繁华破碎，欲觅芳踪而翠钗难卜。欲将旧日宫妆描入画幅，又怕已非如今装束，写出今昔之隔。彩扇红牙等旧物虽在，已无人解听开元盛世之曲。末句以空掩袖、倚寒竹作结，以高寒坚贞之竹自喻，含蓄深沉。

## 女冠子·蕙花香也

**原文：**
> 蕙花香也。雪晴池馆如画。春风飞到，宝钗楼上，一片笙箫，琉璃光射。而今灯漫挂。不是暗尘明月，那时元夜。况年来、心懒意怯，羞与蛾儿争耍。
>
> 江城人悄初更打。问繁华谁解，再向天公借。剔残红灺。但梦里隐隐，钿车罗帕。吴笺银粉砑。待把旧家风景，写成闲话。笑绿鬟邻女，倚窗犹唱，夕阳西下。

**创作背景：**
此词为元宵节追忆故国繁华之作。宋亡之后，词人面对冷落的元宵节，与记忆中南宋临安元宵盛况形成鲜明对比，不胜今昔之感。

**词意简析：**
上片以回忆开篇：昔日元宵，蕙花飘香、雪晴池馆如画，春风中宝钗楼上笙箫齐奏、琉璃灯光四射。而今花灯漫挂，已非当年暗尘随马、明月逐人之盛。年来心懒意怯，再无心情与少女们争耍闹元宵。下片写江城初更人静，问谁能再向天公借回昔日繁华。残灯将灭，钿车罗帕之盛只存于梦中隐约。欲将旧家风景写成吴笺闲话，却见邻家少女倚窗犹唱从前元宵词曲，而不知江山已改。此看似不经意的一笔，反将亡国之痛衬得更加深沉。
