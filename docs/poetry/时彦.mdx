# 时彦

## 词人简介
时彦（？—1107），字邦彦，开封（今属河南）人。宋神宗元丰二年（1079年）状元及第，累官至吏部侍郎。存词仅《青门饮》一首，然此词写边塞行役与离别相思，刚柔相济，情景交融，被选入《宋词三百首》，足以一窥其词才。

---

## 青门饮·胡马嘶风

**原文：**
> 胡马嘶风，汉旗翻雪，彤云又吐，一竿残照。古木连空，乱山无数，行尽暮沙衰草。星斗横幽馆，夜无眠，灯花空老。雾浓香鸭，冰凝泪烛，霜天难晓。
>
> 长记小妆才了，一杯未尽，离怀多少。醉里秋波，梦中朝雨，都是醒时烦恼。料有牵情处，忍思量、耳边曾道。甚时跃马归来，认得迎门轻笑。

**创作背景：**
此词别题《寄宠人》，是时彦远役怀人之作。词人身在边塞荒寒之地，白天行尽暮沙衰草，夜晚独宿幽馆，长夜难眠，于是深深怀念远在故乡的爱人。全词上片写边塞苦寒之景，下片写甜蜜又苦涩的回忆，以壮阔之笔写缠绵之情，是宋词中远役相思的佳作。

**词意简析：**
上片以壮阔苍劲的笔法写边塞行役——胡马嘶风，汉旗翻雪，彤云吐霞，残阳如血。古木连空，乱山无数，日暮时分行尽黄沙衰草。入夜后星斗横斜，独宿幽馆，灯花空老，彻夜难眠，直到霜天难晓。以一系列富有边塞特征的意象，渲染出苍凉、孤寂、漫长的行役之旅。下片笔锋一转，以委婉细腻之笔回忆离别——记得她淡妆完毕，一杯酒未尽，已是离愁满怀。醉中的秋波一瞥、梦中的幽欢蜜爱，醒来后都成了无尽的烦恼。最牵情之处是她附在耳边的那句低语："甚时跃马归来，认得迎门轻笑。"全词上下片刚柔并济，结尾以女子幽微深情的期盼收束，令人回味不尽。
