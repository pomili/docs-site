# 张耒

## 词人简介
张耒（1054—1114），字文潜，号柯山，楚州淮阴（今属江苏）人。"苏门四学士"之一，与黄庭坚、秦观、晁补之齐名。神宗熙宁六年（1073年）进士，累官至起居舍人。因坐元祐党籍，屡遭贬谪，晚年定居陈州。其诗学白居易、张籍，风格平易自然。存词不多，但《风流子》一篇为世所称。

## 风流子·木叶亭皋下

**原文：**
> 木叶亭皋下，重阳近，又是捣衣秋。奈愁入庾肠，老侵潘鬓，谩簪黄菊，花也应羞。楚天晚，白蘋烟尽处，红蓼水边头。芳草有情，夕阳无语，雁横南浦，人倚西楼。
>
> 玉容知安否？香笺共锦字，两处悠悠。空恨碧云离合，青鸟沉浮。向风前懊恼，芳心一点，寸眉两叶，禁甚闲愁？情到不堪言处，分付东流。

**创作背景：**
此词约作于张耒因坐元祐党籍被贬外州期间。重阳将至，词人客居异乡，见木叶凋零、秋风捣衣，勾起了对家乡妻子的深深思念。全词以羁旅之愁与思妻之情交织，是北宋羁旅相思词中的名篇。

**词意简析：**
上片着力写景——树叶飘落水边平地上，重阳将近，又到捣衣时节。词人自叹愁入庾信之肠、衰老如潘岳之鬓，即便勉强插上黄菊，花也应当替他感到羞愧。傍晚极目远望，白蘋烟尽处、红蓼水边头，"芳草有情，夕阳无语，雁横南浦，人倚西楼"四句融情入景，对仗工整，是全词的精华。下片转而想象远方的妻子——书信锦字因两地相隔无法寄达，只能空自怨恨碧云离合、青鸟沉浮。结尾"情到不堪言处，分付东流"以无奈收场，将无尽相思付与滔滔东流之水，沉痛哀婉，意境深远。
