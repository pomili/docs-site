# 周紫芝

## 词人简介
周紫芝（1082—1155），字少隐，号竹坡居士，宣城（今属安徽）人。南宋词人，其词风婉约清丽，语言自然流畅，善于以细腻笔触抒发离愁别恨与身世之感。

---

## 鹧鸪天·一点残红欲尽时

**原文：**
> 一点残红欲尽时，乍凉秋气满屏帏。梧桐叶上三更雨，叶叶声声是别离。
>
> 调宝瑟，拨金猊。那时同唱鹧鸪词。如今风雨西楼夜，不听清歌也泪垂。

**创作背景：**
此词写于秋夜怀人之时。词人独坐灯前，残灯将尽，秋凉透帷，听窗外梧桐夜雨，不禁回忆起当年与意中人调瑟拨香、同唱歌曲的温馨场景，对比如今的孤独凄凉。

**词意简析：**
上片以"一点残红欲尽时"写夜深灯残之景，"乍凉秋气满屏帏"写出秋意透帷的寒凉。"梧桐叶上三更雨，叶叶声声是别离"将雨打梧桐的每一声都听成是离别之音，以物拟情，极为深切。下片转入回忆，"调宝瑟，拨金猊"写昔日的温馨甜蜜，"那时同唱鹧鸪词"呼应词牌，巧妙自然。结句"如今风雨西楼夜，不听清歌也泪垂"今昔对比强烈，不言悲而悲自深。

## 踏莎行·情似游丝

**原文：**
> 情似游丝，人如飞絮。泪珠阁定空相觑。一溪烟柳万丝垂，无因系得兰舟住。
>
> 雁过斜阳，草迷烟渚。如今已是愁无数。明朝且做莫思量，如何过得今宵去。

**创作背景：**
此词写于送别之际。词人在江边送别意中人，心中万般不舍却无可奈何，眼看兰舟远去，愁绪难遣。全词以江边烟柳、斜阳归雁等意象烘托离愁，语浅情深。

**词意简析：**
上片以两个比喻开篇："情似游丝"写情之绵长飘忽，"人如飞絮"写人之漂泊无定，极为贴切。"一溪烟柳万丝垂，无因系得兰舟住"以千丝万缕的柳条比喻挽留之心，却终究系不住离舟，将无奈之感表达得淋漓尽致。下片"雁过斜阳，草迷烟渚"以景写情，愁已无数。末二句"明朝且做莫思量，如何过得今宵去"直白如话却说尽了离别之人最切实的痛苦——不必说明天，连今夜如何度过都是难题。语浅情深，余韵无穷。
