# 晁冲之

## 词人简介
晁冲之（生卒年不详），字叔用，一字用道，济州巨野（今属山东）人。晁补之的从弟，"苏门四学士"中的晁补之、张耒等与他交好。哲宗绍圣初年，新党上台，旧党人物纷纷遭贬，晁冲之隐居河南具茨山，自号具茨先生。其诗属江西诗派，有《晁具茨先生诗集》。词清丽婉约，以淡雅含蓄见长。

## 临江仙·忆昔西池池上饮

**原文：**
> 忆昔西池池上饮，年年多少欢娱。别来不寄一行书。寻常相见了，犹道不如初。
>
> 安稳锦衾今夜梦，月明好渡江湖。相思休问定何如。情知春去后，管得落花无？

**创作背景：**
此词约作于绍圣二年（1095年）前后。晁冲之与苏轼、苏辙及四学士等人交好，昔日常在京师金明池（西池）聚会饮酒。哲宗绍圣初年新党执政，旧党遭贬，晁冲之也被迫离京隐居。独居山中，追忆当年欢聚，感叹朋友离散、人事变迁，以淡雅笔触寄托沉痛的思念。

**词意简析：**
上片追忆往昔——回想当年在西池上宴饮，年年有多少欢乐。然而分别之后，连一行书信也不曾寄来。即使偶尔相见，彼此间也冷冷淡淡，再也回不到当初那种亲密无间的状态。平淡的话语中藏着深重的伤感。下片转为写今夜——安稳卧于锦被中的梦，趁着明月飞渡江湖去与故人相会。但即便在梦中相逢，也不要问彼此的近况如何，"情知春去后，管得落花无"——明明知道春天已经过去，哪里还管得了落花的命运呢？以落花喻指遭受政治风雨摧残的故旧朋友们，语淡意浓，令人叹惋。
