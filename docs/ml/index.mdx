## 学习建议

1. **先过数学基础**，不需要全部记住，了解核心概念即可，遇到不懂的再回头查
2. **按顺序学习监督学习算法**，从线性回归开始，逐步深入
3. **每个算法都动手写代码**，文档中的代码示例建议亲自跑一遍
4. **用实战项目检验学习效果**，理论和手写代码之间还有工程落地这道坎
