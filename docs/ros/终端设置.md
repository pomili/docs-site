# terminator

### 安装

~~~
sudo apt install terminator
~~~

水平分割

```
ctrl+shift+E
```

竖直分割

```
ctrl+shift+O
```

关闭子终端

```
ctrl+shift+W
```

子终端全屏显示

```
ctrl+shift+X
```

### 添加环境变量

把当前工作空间的包加入 ROS 2 环境变量中，让系统能找到你的包、节点、launch 文件等

```
source install/setup.bash
```

或者在 .bashrc 最后一行添加：

```
source ~/ros2_ws/install/setup.bash
```
